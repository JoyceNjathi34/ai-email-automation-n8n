# AI Email Automation Agent (n8n)

This project is an AI-powered email automation workflow built with n8n.

## Features
- Classifies emails (High / Medium / Low priority)
- Generates draft replies
- Automates email organization
- Reduces manual email management

## Tech Stack
- n8n
- AI model
- Gmail API

## Workflow
Import the JSON file into n8n to run the automation.
